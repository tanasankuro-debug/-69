### เกี่ยวกับโครงงาน
เว็บไซต์นี้จัดทำขึ้นเพื่อนำเสนอโครงงานวิจัยเกี่ยวกับการเกิดภาวะ "Stagflation" ในประเทศไทย โดยวิเคราะห์ความขัดแย้งในระบบเศรษฐกิจ:
- **ฝั่งผู้ผลิต (Cost-Push Inflation):** ต้นทุนการผลิตสูงขึ้น ทำให้ต้องปรับราคาสินค้าขึ้น
- **ฝั่งผู้บริโภค (Demand-Side Deflation):** กำลังซื้อลดลง หนี้ครัวเรือนสูง ทำให้ชะลอการใช้จ่าย
