# StagflationTH 📊

โครงงานวิจัยเศรษฐศาสตร์: **ภาวะสองขั้ว เงินเฟ้อต้นทุนของผู้ผลิตและเงินฝืดอุปสงค์ของผู้บริโภคในเศรษฐกิจไทย** (Stagflation)

---

### 📖 เกี่ยวกับโครงงาน
เว็บไซต์นี้จัดทำขึ้นเพื่อนำเสนอโครงงานวิจัยเศรษฐศาสตร์เกี่ยวกับการเกิดภาวะ "Stagflation" ในประเทศไทย โดยนำเสนอความขัดแย้งในระบบเศรษฐกิจผ่านการใช้รูปแบบเว็บแบบ Interactive:
- **ฝั่งผู้ผลิต (Cost-Push Inflation):** ต้นทุนพลังงาน วัตถุดิบ และค่าแรงที่สูงขึ้น บีบให้ต้องปรับราคาสินค้า
- **ฝั่งผู้บริโภค (Demand-Side Deflation):** กำลังซื้อลดลง หนี้ครัวเรือนพุ่งสูง ทำให้ประชาชนชะลอการใช้จ่าย

### ✨ ฟีเจอร์ของเว็บไซต์
- 🎨 **Premium Aesthetic:** ดีไซน์สไตล์ Dark Theme ล้ำสมัย
- 🎬 **Animations:** แอนิเมชันตอนเลื่อนหน้าจอที่ไหลลื่น
- 📈 **Dynamic Charts:** กราฟข้อมูลจริงเปรียบเทียบ CPI, PPI, และหนี้ครัวเรือน ออกแบบด้วย Chart.js
- 📱 **Responsive Design:** แสดงผลได้อย่างสมบูรณ์แบบทั้งบนคอมพิวเตอร์และโทรศัพท์มือถือ

เนื่องจากตัวเว็บไซต์อาจจะยังรันบน GitHub Pages ไม่สมบูรณ์ คุณสามารถเปิดดูบนเครื่องตัวเองได้ง่ายๆ ดังนี้:

3. ดับเบิลคลิกเปิดไฟล์ `index.html` บนเบราว์เซอร์ (Google Chrome, Edge หรือ Safari) ก็จะเห็นหน้าเว็บทั้งหมดทันที
