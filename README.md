### 📖 เกี่ยวกับโครงงาน
เว็บไซต์นี้จัดทำขึ้นเพื่อนำเสนอโครงงานวิจัยเศรษฐศาสตร์เกี่ยวกับการเกิดภาวะ "Stagflation" ในประเทศไทย โดยนำเสนอความขัดแย้งในระบบเศรษฐกิจผ่านการใช้ Interactive Storytelling:
- **ฝั่งผู้ผลิต (Cost-Push Inflation):** ต้นทุนพลังงาน วัตถุดิบ และค่าแรงที่สูงขึ้น บีบให้ต้องปรับราคาสินค้า
- **ฝั่งผู้บริโภค (Demand-Side Deflation):** กำลังซื้อลดลง หนี้ครัวเรือนพุ่งสูง ทำให้ประชาชนชะลอการใช้จ่าย

### ✨ ฟีเจอร์เด่นของเว็บไซต์ (เวอร์ชัน Revamp)
- 🎨 **Premium Aesthetic:** ดีไซน์สไตล์ Dark Theme ล้ำสมัย ผสมผสาน Glassmorphism และ Neumorphism
- 🎬 **Advanced Animations:** ขับเคลื่อนด้วย GSAP ควบคุมแอนิเมชันตอนเลื่อนหน้าจอ (Scroll Reveals) และเอฟเฟกต์แบบ Parallax
- 🕹️ **Micro-Interactions:** เมาส์วงกลมเรืองแสงแบบ Custom (Custom Cursor) และปุ่มที่ตอบสนองต่อการขยับเมาส์ (Magnetic Buttons)
- 📈 **Dynamic Charts:** กราฟข้อมูลจริงเปรียบเทียบ CPI, PPI, และหนี้ครัวเรือน ออกแบบด้วย Chart.js ที่สวยงาม
- 📱 **Responsive Design:** แสดงผลได้อย่างสมบูรณ์แบบทั้งบนคอมพิวเตอร์และโทรศัพท์มือถือ

### 💻 เทคโนโลยีที่ใช้ (Tech Stack)
* **HTML5** & **Vanilla CSS** (No CSS Frameworks)
* **JavaScript** (ES6+)
* **[GSAP](https://greensock.com/gsap/)** (ScrollTrigger) สำหรับแอนิเมชันลื่นไหล
* **[Chart.js](https://www.chartjs.org/)** สำหรับการนำเสนอข้อมูลกราฟ
